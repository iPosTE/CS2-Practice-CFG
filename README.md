# CS2-Practice-CFG

Basic CS2 Practice Config for Premier Maps

## Installation

1. Simply drag all the files to the path folder: ```...\Steam\steamapps\common\Counter-Strike Global Offensive\game\csgo\cfg```,
2. Load any **Premier map**,
3. In console, type: ```exec o_practicebinds```

## Keybind Customization

You can change the shortcut binds in the ```o_practicebinds.cfg``` file
